1、默认是double类型，如：3.1456
2、double类型可以 以d 或 D结尾，也可以不用；
3、float类型必须f 或 F结尾；
4、float类型值赋值给double类型变量，精度还是float类型精度，如：
	double v6 = 3.143424222909089208490284092F; ——> 值为：3.1434242725372314
	double v5 = 3.143424222909089208490284092; ——> 值为：3.1434242229090894
5、有效位数，float类型6~7位，double类型15位
	float v10 = 3.634058_19839508329583095390580985309805080935F; ——> 值为：3.634058_2  有效位小数点后6位
	double v11 = 3.634058198395083_29583095390580985309805080935D; ——> 值为：3.634058198395083_5 有效位小数点后15位
6、float 4字节，double 8字节

8、变量
Java 中“ 字母” 和“ 数字” 的范围更大。字母包括’A’ ~ ’Z’、’a1 ~ ’z1、或在某种语言中表示字母的任何Unicode 字符（含中文）
同样， 数字包括'0' ~ '9’ 和在某种语言中表示数字的任何Unicode 字符
变量名以字母开头，由字母数字组成
变量名大小写敏感，区分大小写的

9、
方法变量必须初始化，否则不能使用，
类变量可不用初始化，创建类时会给变量赋初始值

10、常量 
类常量，以final修饰
方法常量，以static final修饰
		final int D = 12;
		System.out.println(D);
		D = 13; // 编译报错
常量只能赋值一次
